# 四則運算測驗系統使用說明

## 📋 系統概述

本系統提供四種獨立的數學運算測驗功能，每種運算都有專屬的題目生成頁面，支援垂直排列格式的題目卷和答案卷列印。

---

## 🚀 功能特色

### ✅ 四種運算類型
1. **加法運算** - 垂直排列的多個加數運算
2. **減法運算** - 垂直排列的多個減數運算，可設定是否允許負數結果
3. **乘法運算** - 垂直排列的兩個數字相乘
4. **除法運算** - 支援整除或有餘數的除法運算

### ✅ 彈性設定
- 題數可調整（預設18題）
- 數字位數可調整（預設3位數）
- 結果位數限制
- 每種運算都有專屬的進階設定

### ✅ 列印功能
- 題目卷列印（不含答案）
- 答案卷列印（含答案）
- 支援一頁顯示多題（3欄式排版）

## 📝 使用流程

### 步驟 1：選擇測驗類型
根據需要，訪問對應的測驗頁面：
- 加法測驗：`/ExamMath/AdditionTest`
- 減法測驗：`/ExamMath/SubtractionTest`
- 乘法測驗：`/ExamMath/MultiplicationTest`
- 除法測驗：`/ExamMath/DivisionTest`

### 步驟 2：設定參數
在測驗頁面上填寫以下資訊：
1. **學生 ID**（必填）
2. **題數**（可調整，預設18題）
3. **數字位數**（可調整，預設3位數）
4. **其他進階設定**（依各測驗類型而定）

### 步驟 3：產生題目
點擊「🎲 產生題目」按鈕，系統會：
1. 驗證輸入參數
2. 產生符合條件的題目
3. 儲存到資料庫
4. 顯示題目內容

### 步驟 4：列印
題目產生後，可以：
- 點擊「🖨️ 列印題目卷」按鈕列印不含答案的題目
- 點擊「🖨️ 列印答案卷」按鈕列印含答案的題目

## 💾 資料儲存說明

### 題目命名規則
系統會自動產生課程名稱（ClassName），格式如下：

| 運算類型 | 命名格式 | 範例 |
|---------|---------|------|
| 加法 | `Addition_{位數}digit_{時間戳}` | `Addition_3digit_20250103_143052` |
| 減法 | `Subtraction_{位數}digit_{時間戳}` | `Subtraction_3digit_20250103_143052` |
| 乘法 | `Multiplication_{被乘數}x{乘數}_{時間戳}` | `Multiplication_2x2_20250103_143052` |
| 除法 | `Division_{被除數}div{除數}_{類型}_{時間戳}` | `Division_3div1_exact_20250103_143052` |

### 去重機制
- 系統會檢查是否已經產生過相同設定的題目
- 如果已存在，會直接載入之前的題目
- 避免重複產生相同的考券

### 資料庫欄位
題目會儲存在 `Vocabulary` 資料表中：
- **CategoryType**: `Addition`、`Subtraction`、`Multiplication`、`Division`
- **Question**: 題目內容（垂直排列格式，使用 `\n` 分隔）
- **Answer**: 答案
- **ClassName**: 課程名稱
- **TestType**: `Math`

## 🎯 題目生成演算法說明

### 加法題目生成
1. 根據數字位數範圍，隨機產生指定數量的加數
2. 計算總和
3. 檢查總和是否符合結果位數限制
4. 如果不符合，重新產生（最多嘗試100次）
5. 產生垂直排列格式

**智能調整機制：**
如果無法產生符合條件的題目，系統會自動降低數字範圍以確保結果符合限制。

### 減法題目生成
1. 產生第一個數字作為被減數
2. 產生指定數量的減數
3. 計算差值
4. 檢查差值是否符合結果位數限制和正負數設定
5. 如果不符合，調整策略重新產生

**特殊處理：**
- 如果不允許負數，系統會確保被減數大於所有減數的總和
- 如果允許負數，系統會放寬限制

### 乘法題目生成
1. 根據被乘數和乘數的位數範圍，隨機產生兩個數字
2. 計算乘積
3. 檢查乘積是否符合結果位數限制（至少2位數）
4. 如果不符合，調整數字範圍

**優化策略：**
- 使用平方根估算來調整數字範圍
- 確保結果至少為2位數（符合乘法運算特性）

### 除法題目生成

#### 整除模式
1. 先隨機產生除數
2. 隨機產生商（quotient）
3. 計算被除數 = 除數 × 商
4. 確保被除數符合位數要求

#### 有餘數模式
1. 根據被除數位數範圍，隨機產生被除數
2. 隨機產生除數
3. 計算商和餘數
4. 如果餘數為0，調整被除數使其產生餘數

## ⚠️ 注意事項

### 1. 學生 ID 必填
在產生題目之前，必須先輸入學生 ID，否則會顯示錯誤訊息。

### 2. 參數驗證
系統會驗證所有輸入參數：
- 題數必須在 1-100 之間
- 數字位數必須在合理範圍內
- 結果位數限制必須大於等於輸入位數

### 3. 除法特殊限制
- 除數位數不能大於或等於被除數位數
- 除數不能為 0
- 整除模式確保沒有餘數
- 有餘數模式確保有餘數

### 4. 資料庫連線
系統使用現有的資料庫架構，確保：
- 使用 `using` 語句管理資料庫連線
- 避免分散式交易
- 使用相同的連線字串

### 5. 瀏覽器相容性
建議使用以下瀏覽器以獲得最佳體驗：
- Chrome 90+
- Edge 90+
- Firefox 88+
- Safari 14+

## 🐛 常見問題排解

### Q1: 無法產生題目？
**A:** 檢查以下項目：
1. 是否已輸入學生 ID
2. 參數設定是否合理（例如：結果位數是否足夠大）
3. 網路連線是否正常
4. 瀏覽器控制台是否有錯誤訊息

### Q2: 題目重複？
**A:** 系統會檢查是否已經產生過相同設定的題目。如果需要產生新的題目，請：
1. 調整參數設定（例如：改變題數或位數）
2. 等待一段時間後重新產生（系統會依據時間戳區分）

### Q3: 列印格式不正確？
**A:** 請檢查：
1. 瀏覽器列印設定中的「縮放」選項（建議 100%）
2. 紙張大小設定（建議 A4）
3. 邊距設定（建議使用預設值）
4. 是否啟用「背景圖形」選項

### Q4: 題目太簡單或太難？
**A:** 可以調整以下參數：
1. 增加或減少數字位數
2. 增加或減少加數/減數/乘數的數量
3. 調整結果位數限制
4. 對於除法，可以調整除數位數
