## 🎭 FlashWolf 招牌重新設計

### 問題：原設計過白，缺乏招牌質感

### 解決方案：

#### 1. **顏色調整** 💙
```css
/* 從純白色改為帶藍色調的淡色 */
color: #e0f7ff;  /* 不再是 #ffffff */
```

#### 2. **霓虹燈管描邊效果**
```css
-webkit-text-stroke: 2px #00d4ff;
```
- 模擬真實霓虹燈管的邊框

#### 3. **多層發光系統**
- **內層**：淺藍強光 (#b3e5ff, #80d4ff)
- **中層**：霓虹藍光 (#00d4ff)
- **外層**：擴散光暈 (#0099ff, #0066ff)
- **遠距離**：大範圍柔和光暈

#### 4. **真實閃爍動畫**
```css
/* 模擬霓虹燈管的不穩定電流 */
animation: neonFlicker 3.5s ease-in-out infinite alternate;
```
- 正常發光 → 突然變暗 → 恢復發光
- 隨機時間點閃爍

#### 5. **字體強化**
- 字體大小：58px → 72px
- 字重：加粗至 900
- 字距：3px → 5px
- 斜體 + 傾斜效果保持

## 🔧 技術細節

### CSS 特效技術

#### 1. **Flexbox 布局**
```css
.login-container {
    display: flex;
    align-items: stretch;  /* 左右等高 */
}
```

#### 2. **多層偽元素背景**
- `::before` - 動態點陣背景
- `::after` - 中央光暈效果

#### 3. **filter: drop-shadow** (圖片用)
```css
filter: drop-shadow(0 0 25px rgba(0, 212, 255, 0.9))
        drop-shadow(0 0 50px rgba(0, 212, 255, 0.7))
        drop-shadow(0 0 80px rgba(0, 153, 255, 0.5));
```

#### 4. **text-shadow** (文字用)
```css
/* 9 層陰影，模擬真實光暈 */
text-shadow: 
    0 0 5px #b3e5ff,
    0 0 10px #80d4ff,
    /* ... 省略其他層 */
    0 0 150px rgba(0, 102, 255, 0.5);
```

#### 5. **動畫時間軸**
```css
/* 閃爍動畫關鍵幀 */
0%, 18%, 22%, 25%, 53%, 57%, 100%  /* 正常發光 */
20%, 24%, 55%                        /* 瞬間變暗 */
```

### 目前狀態
- **文字版招牌**：已啟用，可直接查看效果
- **圖片版招牌**：等待圖片放置

### 如何切換到圖片版本
1. 準備 FlashWolf 霓虹燈圖片
2. 命名為 `flashwolf-neon.png`
3. 放置到 `ERP.Web/wwwroot/images/`
4. 重新整理頁面即可看到圖片版本

- **尺寸**：1400 x 400 px

- **顏色**：藍色霓虹燈效果
- **風格**：類似提供的 "OPEN" 招牌照片

## 📊 效能考量

### 優化措施
1. **CSS 動畫**：使用 GPU 加速的 transform 和 opacity
2. **偽元素**：用於裝飾效果，避免額外 DOM
3. **圖片懶載入**：onerror 備援機制
4. **響應式設計**：@media query 優化小螢幕

### 效能指標
- 首次載入：< 2 秒
- 動畫流暢度：60 FPS
- 適配裝置：桌面、平板、手機
