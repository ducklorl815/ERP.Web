## 📋 更新日期

## 🌙 動畫概念

全新的「迷濛夢幻」霓虹燈動畫效果，模擬真實霓虹燈管在夜晚的呼吸感：
- **不再持續閃爍**
- **加入 blur（模糊）效果**
- **慢慢產生迷濛感**
- **漸進暗淡**
- **最後一次閃爍**

## 💡 關鍵技術

### 1. **Filter Blur**
```css
filter: blur(2px);
```
- 創造模糊迷濛效果
- 最大 2px，最小 0px
- 漸進變化，不突兀

### 2. **Opacity 透明度**
```css
opacity: 0.4;
```
- 控制整體明暗度
- 最暗 0.4，最亮 1.0
- 與 blur 搭配產生深度

### 3. **Text-Shadow / Drop-Shadow**
```css
/* 文字版本 */
text-shadow: 0 0 20px #00d4ff, ...;

/* 圖片版本 */
filter: drop-shadow(0 0 20px rgba(0, 212, 255, 0.8)) ...;
```
- 多層陰影創造光暈
- 範圍從 30px 到 240px
- 隨時間動態變化

### 4. **Color 顏色變化**
```css
/* 亮 → 暗 */
#e0f7ff → #d0ecff → #c0e4ff → ... → #70b0d0
```
- 顏色隨亮度漸變
- 保持藍色調統一
- 增加真實感

## 🔧 調整參數

### 如何調整模糊強度

**更模糊（夢幻感更強）：**
```css
70% {
    filter: blur(3px);  /* 原本 2px */
}
```

**較清晰（保守版）：**
```css
70% {
    filter: blur(1px);  /* 原本 2px */
}
```

### 如何調整暗淡程度

**更暗：**
```css
90% {
    opacity: 0.3;  /* 原本 0.4 */
}
```

**較亮：**
```css
90% {
    opacity: 0.5;  /* 原本 0.4 */
}
```

### 如何調整循環時間

**更慢（更舒緩）：**
```css
animation: neonDreamGlow 16s ...;  /* 原本 12s */
```

**更快（更頻繁）：**
```css
animation: neonDreamGlow 8s ...;  /* 原本 12s */
```

### 如何調整閃爍時機

**更早閃爍：**
```css
85% { /* 最暗狀態 */ }
86% { /* 閃爍 */ }
87% { /* 恢復 */ }
```

**更晚閃爍（戲劇性更強）：**
```css
93% { /* 最暗狀態 */ }
94% { /* 閃爍 */ }
95% { /* 恢復 */ }
```

## 📐 與其他動畫的整合

### 頁面整體動畫層次

1. **霓虹燈招牌**：12s 迷濛循環（主角）
2. **背景點陣**：30s 緩慢移動（襯托）
3. **狼爪圖示**：2s 脈動（裝飾）
4. **輸入框焦點**：0.4s 快速反饋（交互）
5. **按鈕懸停**：0.4s 快速反饋（交互）

### 時間協調

所有動畫採用不同週期，避免同步：
- 12s（霓虹燈）
- 30s（背景）
- 2s（脈動）

創造豐富的視覺層次，不會重複單調。

## 🚀 效能考量

### 效能優化

| 技術 | 效能影響 | 說明 |
|------|---------|------|
| `filter: blur()` | 🟡 中等 | GPU 加速，可接受 |
| `drop-shadow` | 🟡 中等 | 多層會增加負擔 |
| `text-shadow` | 🟢 低 | CSS 原生，效能好 |
| `opacity` | 🟢 低 | GPU 加速 |
| 關鍵幀數量 | 🟢 低 | 10 個關鍵幀適中 |

### 最佳化建議

1. **在低效能裝置上：**
   ```css
   /* 減少 blur 強度 */
   filter: blur(1px);  /* 原本 2px */
   
   /* 減少 drop-shadow 層數 */
   filter: drop-shadow(0 0 20px ...) 
           drop-shadow(0 0 40px ...);
   /* 移除第三層 */
   ```

2. **省電模式：**
   ```css
   /* 延長動畫時間 */
   animation: neonDreamGlow 20s ...;
   ```

3. **移除動畫：**
   ```css
   /* 完全停用動畫 */
   animation: none;
   ```
