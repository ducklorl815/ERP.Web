## ⚡ 霓虹燈閃爍頻率調整說明

### 1. **調整位置**
在 `Login.cshtml` 的 CSS 區域找到：
```css
animation: neonFlicker 10s ease-in-out infinite alternate;
```

### 2. **調整方法**

#### 方法一：改變整體循環時間
```css
/* 目前設定 */
animation: neonFlicker 10s ease-in-out infinite alternate;

/* 閃爍更快 */
animation: neonFlicker 5s ease-in-out infinite alternate;

/* 閃爍更慢 */
animation: neonFlicker 15s ease-in-out infinite alternate;

/* 非常快速 */
animation: neonFlicker 2s ease-in-out infinite alternate;
```

**說明：**
- `10s` = 整個動畫循環的時間
- 時間越短 = 閃爍越快
- 時間越長 = 閃爍越慢

#### 方法二：調整閃爍時間點（百分比）

在 `@@keyframes neonFlicker` 區塊中：

```css
@@keyframes neonFlicker {
    /* 正常發光狀態 */
    0%, 18%, 22%, 25%, 53%, 57%, 100% {
        color: #e0f7ff;
        text-shadow: ... /* 完整發光效果 */
    }
    
    /* 閃爍變暗狀態 */
    20%, 24%, 55% {
        color: #a0c8e0;
        text-shadow: ... /* 減弱的發光效果 */
    }
}
```

**調整說明：**

**增加閃爍次數：**
```css
/* 原本 2 次閃爍 */
20%, 24%, 55%

/* 改成 3 次閃爍 */
20%, 24%, 50%, 54%, 75%, 79%
```

**調整閃爍持續時間：**
```css
/* 短暫閃爍（快閃） */
20%, 21%  /* 只閃 1% 的時間 */

/* 中等閃爍 */
20%, 24%  /* 閃 4% 的時間（目前設定）*/

/* 持續閃爍 */
20%, 30%  /* 閃 10% 的時間 */
```

### 3. **完整範例**

#### 例子 A：快速頻繁閃爍
```css
/* 2秒循環，3次閃爍 */
animation: neonFlicker 2s ease-in-out infinite alternate;

@@keyframes neonFlicker {
    0%, 18%, 22%, 25%, 43%, 47%, 50%, 68%, 72%, 75%, 100% {
        /* 正常發光 */
    }
    20%, 21%, 45%, 46%, 70%, 71% {
        /* 快速閃爍 */
    }
}
```

#### 例子 B：慢速單次閃爍
```css
/* 8秒循環，1次閃爍 */
animation: neonFlicker 8s ease-in-out infinite alternate;

@@keyframes neonFlicker {
    0%, 48%, 52%, 100% {
        /* 正常發光 */
    }
    50% {
        /* 中間閃一次 */
    }
}
```

#### 例子 C：不穩定閃爍（推薦真實感）
```css
/* 目前使用：10秒循環，2次不規則閃爍 */
animation: neonFlicker 10s ease-in-out infinite alternate;

@@keyframes neonFlicker {
    0%, 18%, 22%, 25%, 53%, 57%, 100% {
        /* 正常發光 */
    }
    20%, 24%, 55% {
        /* 變暗 */
    }
}
```

### 4. **圖片版本的閃爍調整**

圖片版本使用不同的動畫名稱：
```css
animation: neonImageGlow 3.5s ease-in-out infinite alternate;

@@keyframes neonImageGlow {
    0%, 19%, 21%, 23%, 25%, 54%, 56%, 100% {
        filter: drop-shadow(...);
        opacity: 1;
    }
    20%, 24%, 55% {
        filter: drop-shadow(...);
        opacity: 0.85;
    }
}
```

調整方法與文字版本相同。

## 💡 使用建議

### 閃爍頻率設定建議

**辦公環境：**
```css
animation: neonFlicker 10s ease-in-out infinite alternate;
```
- 理由：較慢的閃爍不會分散注意力

**展示/行銷：**
```css
animation: neonFlicker 3s ease-in-out infinite alternate;
```
- 理由：較快的閃爍更吸引眼球

**極簡風格：**
```css
animation: neonFlicker 20s ease-in-out infinite alternate;
```
- 理由：極慢的閃爍，接近靜態

**無閃爍：**
```css
/* 移除 animation 行即可 */
```
- 理由：完全靜態的霓虹燈效果

## 📞 常見問題

### Q1: 如何完全停止閃爍？
**A:** 在 `.brand-name-text` 樣式中，移除或註解掉這一行：
```css
/* animation: neonFlicker 10s ease-in-out infinite alternate; */
```

### Q2: 可以讓圖片版和文字版閃爍速度不同嗎？
**A:** 可以！它們使用不同的動畫：
- 文字版：`neonFlicker 10s`
- 圖片版：`neonImageGlow 3.5s`

獨立調整即可。

### Q3: 如何修改左側說明的功能清單？
**A:** 在 HTML 中找到 `.feature-list`，增加或修改 `<li>` 項目：
```html
<ul class="feature-list">
    <li class="feature-item">您的功能描述</li>
    <!-- 可以增加更多項目 -->
</ul>
```

### Q4: 閃電符號可以改成其他圖示嗎？
**A:** 可以！在 CSS 中修改：
```css
.feature-item::before {
    content: '✓';  /* 或 '●', '▶', '★' 等 */
}
```

### Q5: 可以調整招牌在右側的垂直位置嗎？
**A:** 可以！調整 `.brand-header` 的 padding：
```css
.brand-header {
    padding: 30px 50px 20px;  /* 上 左右 下 */
}
```
