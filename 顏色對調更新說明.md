## 🌙 右側 - 登入區域（改為深色）

### 背景
```css
background: linear-gradient(180deg, #1a1a2e 0%, #0a0a0a 100%);
```

### 特殊效果
- 中央藍色光暈（營造霓虹燈氛圍）
- 半透明層次感
- 招牌區與表單區漸層區分

### 招牌區樣式
```css
.brand-header {
    background: linear-gradient(180deg, 
        rgba(26, 26, 46, 0.8) 0%, 
        rgba(26, 26, 46, 0.4) 100%);
    border-bottom: 2px solid rgba(0, 212, 255, 0.2);
}
```

### 文字顏色
| 元素 | 顏色 | 說明 |
|------|------|------|
| 標題 | `#e0f7ff` | 淡藍白色（帶發光） |
| 副標題 | `#80d4ff` | 淺藍色 |
| 標籤 | `#b3d9f0` | 中淺藍色 |
| 輸入框文字 | `#e0f7ff` | 淡藍白色 |
| Placeholder | `rgba(128, 212, 255, 0.5)` | 半透明藍色 |
| 記住我 | `#b3d9f0` | 中淺藍色 |
| 頁尾 | `#6b9cb8` | 灰藍色 |

### 輸入框樣式
```css
.form-control {
    background-color: rgba(26, 26, 46, 0.5);
    border: 1px solid rgba(0, 212, 255, 0.3);
    color: #e0f7ff;
}

.form-control:focus {
    background-color: rgba(26, 26, 46, 0.7);
    border-color: #00d4ff;
    box-shadow: 0 0 0 0.2rem rgba(0, 212, 255, 0.25);
}
```

## 🔍 技術細節

### 深色背景的半透明輸入框
```css
/* 讓輸入框有玻璃質感 */
background-color: rgba(26, 26, 46, 0.5);

/* 焦點時加深背景 */
.form-control:focus {
    background-color: rgba(26, 26, 46, 0.7);
}
```

### 發光邊框效果
```css
/* 正常狀態 - 淡藍色半透明邊框 */
border: 1px solid rgba(0, 212, 255, 0.3);

/* 焦點狀態 - 亮藍色實線邊框 + 外發光 */
.form-control:focus {
    border-color: #00d4ff;
    box-shadow: 0 0 0 0.2rem rgba(0, 212, 255, 0.25);
}
```

### 文字對比度
- 左側：深色文字 (#1a1a2e, #555, #666) on 白色背景 → 對比度高
- 右側：淺色文字 (#e0f7ff, #b3d9f0) on 深色背景 → 對比度高
